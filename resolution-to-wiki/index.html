Resolutions Favelet

This is a "favelet" to generate a rendered preview of content in a GitHub repository. To install it, use the context menu or drag the following link to your bookmarks toolbar or menu. When viewing a file in GitHub, activate this favelet to the rendered preview.

Resolutions

var resolutions = document.querySelector('#ResolutionSummary').parentNode.nextElementSibling.getElementsByTagName('a'); if (!resolutions.length > 0) alert ('No resolutions found in these minutes.'); var pathArray = window.location.pathname.split(new RegExp('[/-]')); var date = pathArray.slice(1, 4).join('-'); var output = ''; for (i = 0; i < resolutions.length; i++) { item = resolutions[i]; output += date + ': ' + '[' + item.href + ' ' + item.innerHTML.replace(/\n/, ' ') + ']\n'; }; alert (output);
